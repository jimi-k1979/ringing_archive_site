Call Change Compositions

This section contains all sorts of call change compositions which I've either found in various places or composed myself. Each composition can be downloaded as a pdf file in two forms: one ready for the conductor in the tower with just the changes written on it and the other with the resulting order of each change written after the change. The long format also has named changes noted where applicable.

All changes are given as if calling up (i.e. the first bell moves up after the second bell). If your preference is for calling the changes down (the second bell moves down behind the first bell, there is a subtle difference!) look at the long form of the changes to convert them.

To view or download the compositions either right click on an icon and select 'Save target as...' to download, or left click an icon to view the composition. If you don't already have a pdf reader installed on your computer it can be downloaded from the Adobe website by clicking on the icon below.
